# Welcome to the repository for my personal webpage

This webpage is built with:

- [Hugo](https://gohugo.io/) (theme inspiration from: [hugo-profile](https://github.com/gurusabarish/hugo-profile))

The website can be found at [cedric-neergaard.dev](https://cedric-neergaard.dev/). It serves as a personal portfolio and blog, showcasing projects, skills, and experiences.

## Structure

The website is structured as a Hugo project with the following key directories:

- `archetypes`: Defines content archetypes.
- `content`: Contains the website's content, including sections like:
    - About: Information about me.
    - Projects: A showcase of personal projects and contributions.
    - Blog: Articles and posts on various topics.
- `layouts`: Defines the HTML templates for rendering the content. Includes layouts for the homepage (`index.html`), individual pages, and partials for reusable components.
- `static`: Contains static assets such as images, CSS, and JavaScript.
